rrbgen
======
**__Current Version: 0.0.1__**
Release date: June 6, 2018

Robbie's lightweight limited functionality R bgen read/write library

[![Build Status](https://travis-ci.org/rwdavies/rrbgen.svg)](https://travis-ci.org/rwdavies/rrbgen)

This library supports "v1.3" of bgen defined [here](http://www.well.ox.ac.uk/~gav/bgen_format/). It supports reading and writing using 8, 16, 24 or 32 bits per probability, using Layout = 2 and CompressedSNPBlocks = 1. Any other format specifications may crash unexpectedly without properly defined error.
